# One Minute Escapes

7 generative art experiences. Scroll through them like a feed.
Each one loops forever, different every time.
No score, no goal, no point.

**[→ Try it](https://jerrysoer.github.io/one-minute-escapes/)**

## The Escapes

| # | Name | What Happens |
|---|------|-------------|
| 1 | Murmuration | A flock moves as one mind. Your cursor bends them. |
| 2 | Rain | Drops on glass. Rivulets form. The light changes. |
| 3 | Letters | Characters fall like snow. Words emerge from chaos. |
| 4 | Gravity | Orbs orbit, collide, merge. Add more. Break everything. |
| 5 | Erosion | Wind carves stone. A million years in a minute. |
| 6 | Migration | Creatures cross a landscape. Winter becomes spring. |
| 7 | Unfold | Geometric forms breathe and morph. Platonic solids dream. |

Each one is generative — different every time you watch.
Stay on one forever, or scroll through them all.

## Stack

Canvas 2D · Three.js (Unfold only) · React · Vite · GitHub Pages

No backend. No database. No tracking. No cookies.
